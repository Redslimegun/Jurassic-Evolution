# TP03 — Site Web Narratif : Jurassic Evolution

## Description
Jurassic Evolution est un site narratif immersif qui revisite la saga Jurassic Park / World à travers une approche visuelle et interactive. 
L'expérience invite l'utilisateur à explorer l'évolution du mythe des dinosaures, de 1993 à aujourd'hui.

## Instructions
- Ouvrir `main.html` dans un navigateur pour commencer l’aventure.  
- Cliquer sur les options pour naviguer dans le site web.  
- Les images sont placées dans `/images/`.  
- La vidéo est déposée dans `/video/`.
